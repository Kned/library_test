# README
This is a simple library project. I tried to make as simple as possible, but still covering the needed actions to the library to work. Also has an api that allows all functions via api rest.
* Ruby version
*   ruby "3.0.4"

* System dependencies
*   Postgres 16.3

* Setup
*   $bin/setup

* Run project
*   $bin/dev

* Run rspec
*   bundle exec rspec
